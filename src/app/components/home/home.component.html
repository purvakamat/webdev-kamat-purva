Description

The Website we have built throughout the assignments allows users to create and use online, mobile friendly, Websites. It is a Website to create Websites. We will refer to those users creating the Websites as developers. Users that visit the Websites created by the developers will be referred to as end users. Only developers can create and modify the Websites. End users can not modify the Websites, they can only interact with them. The Website is be built using MEAN stack using the four underpinning technologies MongoDB, Express, AngularJS 4, and Node.js.

PROJECT


NU Shuttle

Northeastern University has a shuttle service from dusk to dawn which can be availed by all students to get dropped to their homes within a 2 mile radius of the University. This service is called the RED EYE. The shuttle leaves from Snell Library every 20 minutes and students queue up in front of the library to catch the next shuttle. This becomes difficult when the weather conditions in Boston worsen and students have to wait in queue in the chilling winter. Also, the drivers of the van manually take down the addresses of the drop-off locations from the students and decide on the shortest route for the shuttle. Since this is manually calculated, it may not always be the shortest path and can cause delays. In our project we have solved this problem by creating a virtual queue service which will allow students to queue up for the shuttle without having to physically stand in line.
